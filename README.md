图片要求
--------------------
一个 **故事** 需要1个彩色人物头像, 1个长条banner封面,   
每个情节都 1个剧情的普通图片 和 3个结局的普通图片。

封面尺寸：800 * 450  
头像尺寸：200 * 200  
图片尺寸：640 * 360
